# Protein-Structure-Prediction# Protein Secondary Structure Prediction

## Overview
This project predicts protein secondary structure from amino acid sequences using machine learning.


## How to Run
1. Make sure Python 3 is installed 
2. Install dependencies: pip install scikit-learn numpy
3. Place `pr_data.txt` in the same directory  
4. Run the code: python Protein_Prediction_Model_NB.py


## Output
The program prints the model accuracy on the test dataset.


## Files
- `Protein_Prediction_Model_NB.py` → main implementation  
- `pr_data.txt` → dataset
